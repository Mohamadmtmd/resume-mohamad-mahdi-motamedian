# Resume of Mohammad Mahdi Motamedian

## Overview

This is a personal portfolio/resume web page designed to showcase the skills, experience, and education of Mohammad Mahdi Motamedian, a Network Security Specialist. The page is built with **HTML, CSS, and vanilla JavaScript**, focusing on a clean, responsive design that works well on desktop and smaller screens.

---

## Features & Design Decisions

* **Layout:**

  * Two-column layout for desktop (sidebar + main content).
  * Responsive mobile-friendly design using CSS Flexbox.

* **Sidebar / Profile Section:**

  * Profile image, name, job title, and brief bio.
  * Personal info including birth date, location, education, email, and phone.
  * "Send Message" button triggers a JavaScript alert for contact.

* **Main Content Section:**

  * Introduction with title and subtitle.
  * About Me section with a brief description.
  * Skills section with animated progress bars showing proficiency in different technologies:

    * FortiGate Firewall (NGFW)
    * FortiWeb (WAF)
    * Linux / Server Hardening
    * Cisco (CCNA/CCNP)
    * Network Infrastructure & Monitoring
  * Experience section listing relevant work history.
  * Education section showing academic background.

* **Interactive Features:**

  * Animated skill progress bars when scrolling into view.
  * "Back to Top" button for quick navigation.
  * Contact button triggers an alert with email instructions.

* **Styling:**

  * Clean and professional color scheme with teal highlights (`#007b83`).
  * Rounded cards and smooth transitions for progress bars.
  * Font: Vazirmatn from Google Fonts for Persian readability.

---

## Technologies Used

* **HTML5:** Page structure and semantic markup.
* **CSS3 & Flexbox:** Styling, layout, responsive design, and animations.
* **JavaScript:**

  * Scroll-triggered animations for skill bars.
  * Back to top button behavior.
  * Contact button interaction.

---

## Usage

1. Clone or download the repository.
2. Open `index.html` in any web browser to view the resume.
3. For customization, update the profile information, skills, experience, and education in the HTML and CSS files.

---

## Notes

* Fully responsive layout for both desktop and smaller screens.
* No external frameworks like Bootstrap or jQuery are used.
* Lightweight and simple code for easy maintenance.
